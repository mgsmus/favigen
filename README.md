# Favigen
Favigen is a very simple Python application for creating common favicon and app icons.

# Installation

Use the package manager [pipenv](https://docs.pipenv.org/en/latest/) to install.

```bash
pipenv install
```

## Usage

```python
./favigen.py logo.png
```
